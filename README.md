# 🚢 Our Wedding · Ha Long Bay, Vietnam

An information website for our wedding aboard the **Ambassador Cruise** in Ha Long Bay, Vietnam. Built as a fast, single-page static site (HTML + CSS + vanilla JS) — easy to host anywhere.

## ▶️ Running locally
```
node server.js
```
Then open **http://localhost:8000**. Press `Ctrl+C` to stop.
(No installs required — `server.js` is a tiny zero-dependency static server.)

## 📂 Files
- `index.html` — page content & all sections
- `style.css` — Ha Long Bay coastal design system (teal + gold)
- `app.js` — countdown timer, mobile nav, scroll animations, active-link highlighting
- `server.js` — local preview server

## 🧭 Sections
1. **Hero** — names, location, live countdown
2. **Welcome** — trip overview + quick facts
3. **The Cruise** — Ambassador Cruise details (cabins, onboard, embarkation)
4. **Wedding Itinerary** — day-by-day schedule
5. **Pre-Trip** — Hanoi hotels + transportation
6. **Post-Trip** — your plans + nearby destinations
7. **Travel Essentials** — visa, flights, money, packing, etc.
